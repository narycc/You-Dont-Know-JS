为什么我还要再翻译一次呢？

## 我的目标： 

1，通过翻译这一系列丛书，补充自己对JS 的认知；

2，通过完成这本书，锻炼自己的恒心；

3，通过完成这系列书的翻译，提高自己的翻译水平；

4，最终目标，加入中国开源项目的翻译小组，和更多专业的人一起做有意义的翻译。
